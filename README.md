# Cocktail-RAG
En este proyecto usamos la base "Cocktail Recipes" (ver enlace en el Read Me) para implementar la metodología de Retrieval Augmented Generation (RAG) que toma ingredientes, recetas y datos de los cocteles de la base y propone cocteles para preparar según el input del usuario. Usamos Sentence Transformers como encoder y Mistral 7b como decoder.
